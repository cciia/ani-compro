# вҡҷпёҸ Installation

## 1. Clone Repository


Masuk ke folder project.

```bash

```

---

# рҹ“Ұ Install Dependencies

Install seluruh dependency project.

```bash
npm install
```

---

## Install Package Tambahan

### Next Themes

Digunakan untuk fitur Dark Mode.

```bash
npm install next-themes
```

---

### Next Intl

Digunakan untuk fitur Multi Language.

```bash
npm install next-intl
```

---

### Framer Motion

Digunakan untuk animasi website.

```bash
npm install framer-motion
```

---

### React Icons

Digunakan sebagai library icon.

```bash
npm install react-icons
```
